# A spatiotemporal scaffold in the human subiculum supports episodic memory and its enhancement through navigation training

This repository contains the analysis code and data resources for the paper:
**A spatiotemporal scaffold in the human subiculum supports episodic memory and its enhancement through navigation training** *Jung Han Shin, Verner P. Bingman, and Sang Ah Lee*

## Software requirements
---
- MATLAB 2022a or higher
---
## Script details

### Behavioral scripts (`code_behavior/`)
- **`code1_behavior_overall.m`**  
  Overall behavioral analyses across tasks.

- **`code2_training_spatial.m`**  
  Behavioral effects of navigation training on spatial memory.

- **`code3_training_em.m`**  
  Training-related effects on episodic memory performance.

---

### Neural scripts (`code_neural/`)
- **`code1_act_roi.m`**  
  ROI-based neural activation analysis.

- **`code2_pat_roi.m`**  
  ROI-based neural pattern analysis.

- **`code3_neural_compress.m`**  
  Neural representational compression / dimensionality analysis.

- **`code4_training_act.m`**  
  Training effects on neural activation measures.

- **`code5_training_pat.m`**  
  Training effects on neural pattern reinstatement.

---

## Data
The `data_organized/` directory contains organized data used for analysis.
